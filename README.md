# `NonLatinForSBClassic` Plugin for Squeezebox/Lyrion Music Server

Display non-Latin characters beautifully on your Squeezebox Classic devices.

## Overview

This plugin enables rendering of non-Latin characters on graphics-capable Squeezebox players from the Classic family. This includes support for Chinese, Japanese, Cyrillic, Greek, and extended Latin scripts, allowing you to see artist names, album titles, and track information correctly in any language supported by the font.

### Supported Devices

The plugin works with the following ip3k-based Squeezebox players with graphical displays:

- Squeezebox Transporter
- Squeezebox Boom
- Squeezebox Classic
- Squeezebox 2
- Squeezebox 3
- Squeezebox G

## About the Font

The plugin uses the **Go Noto Universal** font, a carefully optimized subset of Google's [Noto Fonts](https://github.com/notofonts/latin-greek-cyrillic).

This font is derived from the [notofonts/latin-greek-cyrillic](https://github.com/notofonts/latin-greek-cyrillic) project and has been optimized for embedded display systems through the [go-noto-universal](https://github.com/satbyy/go-noto-universal) project.

## Installation

Simply enable the plugin through your Lyrion Music Server settings. The plugin will automatically enable proper rendering of non-Latin scripts on your player displays.
